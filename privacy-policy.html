Privacy Policy

Chrome Extension

Effective Date: July 16, 2026  |  Last Updated: July 16, 2026

1. Introduction

This Privacy Policy describes how PageEraser ("the Extension", "we", "our") handles user data. PageEraser is a Chrome/Edge browser extension that allows users to interactively select and remove HTML elements from web pages. We are committed to protecting your privacy and being transparent about our data practices.

🔒

PageEraser does NOT collect, transmit, or share any personal data.

2. Data Collection

PageEraser does not collect any personal information or user data. Specifically:

  • No personal information (name, email, address, etc.) is collected
  • No browsing history or activity is tracked
  • No analytics or telemetry data is gathered
  • No cookies are set by the extension
  • No data is transmitted to any external server
  • No third-party tracking scripts are used
  • No advertising networks are integrated

3. Local Data Storage

PageEraser stores the following data locally on your device using the Chrome Storage API (chrome.storage.local):

Data Type Purpose Storage Location
CSS selectors of removed elements Persist element removal rules across page reloads Local device only
Extension preferences Store user settings (theme, language, sound on/off) Local device only
Domain-scoped rules Associate removal rules with specific websites Local device only
Important: All data remains entirely on your device. No data is ever sent to any remote server, cloud service, or third party. The extension works fully offline after installation.

4. Permissions Explanation

The extension requires certain browser permissions to function. Here is a transparent explanation of each permission and why it is needed:

Permission Why It's Needed
storage To save your element removal rules and preferences locally on your device
activeTab To interact with the current page when you activate element selection mode
contextMenus To add "Erase Element" and "Stretch Element" options to the right-click menu
<all_urls> To inject content scripts on any webpage so element removal works on all sites you visit

5. Third-Party Services

PageEraser does not use any third-party services, APIs, analytics platforms, or advertising networks. The extension is completely self-contained and operates entirely within your browser.

6. Data Sharing

We do not sell, trade, rent, or otherwise share any user data with third parties. Since no data is collected, there is nothing to share.

7. Data Security

All data stored by PageEraser is protected by Chrome's built-in extension security model. Data stored via chrome.storage.local is:

  • Accessible only to the PageEraser extension
  • Isolated from other extensions and web pages
  • Stored in Chrome's encrypted profile directory
  • Automatically removed when the extension is uninstalled

8. User Rights & Data Deletion

You have full control over your data at all times:

  • View your data: Open the PageEraser popup to see all stored rules for the current site
  • Delete specific rules: Use the "Restore" button next to any rule, or drag it to the Recycle Bin
  • Delete all rules for a site: Click "Restore All Layout" in the popup
  • Export your data: Use File → Export Rules to save all rules as a JSON file
  • Delete all data: Uninstalling the extension automatically removes all stored data

9. Children's Privacy

PageEraser does not knowingly collect any information from children under 13 years of age. The extension does not collect any personal information from any user regardless of age.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last Updated" date. We encourage you to review this policy periodically.
