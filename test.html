Video Resizer Test Cases

Test the extension under containing block and overflow hidden clipping conditions.

Case 1: Overflow Hidden + CSS Transform (translate3d, scale)

The container forces a containing block. If containing blocks are not broken, the popped-out fixed video will clip or misalign.

Case 2: Overflow Hidden + Clip Path + CSS Filter

The container limits rendering boundary. Without style override, resizing the video will keep it clipped within the red boundary.
